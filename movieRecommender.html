The Movie Recommender is an algorithm that, based on different components, returns a list of films that best suits the needs of the user. The recommender uses these following elements:

  • Genre
  • Year
  • Rating
  • Awards
  • MPAA

Using these five categories, the algorithm goes through the film database and chooses a list of films that best suit the user's choices.
